#  Problem Statement

Managing daily expenses is a common challenge faced by individuals, especially students and working professionals. Most people rely on manual methods or basic applications that only record expenses without providing intelligent insights. These methods require users to manually categorize each expense, making the process time-consuming, error-prone, and inconsistent.

Additionally, users often fail to analyze their spending patterns due to the lack of automation and smart features in traditional expense tracking systems. Without proper categorization and analysis, it becomes difficult to control unnecessary spending and make informed financial decisions.

#  Objective

The objective of this project is to develop an AI-Powered Expense Tracker that simplifies expense management using Machine Learning techniques. The system aims to:
- Record daily expenses efficiently
- Automatically categorize expenses using a Machine Learning model
- Reduce manual effort and improve accuracy
- Provide a clear summary of total spending

# Proposed Solution

This project proposes an intelligent expense tracking system that uses a Machine Learning model (Naive Bayes) to automatically categorize expenses based on user input. The system processes the expense name, predicts its category, and stores the data along with the amount.

By integrating AI into a simple expense tracker, the application enhances usability, reduces user effort, and provides a smarter way to manage personal finances.
