# AI Expense Tracker

An intelligent expense tracker that uses Machine Learning to automatically categorize expenses.

---

## Features
- Add daily expenses
- Automatic category prediction using AI
- View total spending
- Simple and easy-to-use interface

---

## Tech Stack
- Python
- Scikit-learn
- Naive Bayes Algorithm

---
